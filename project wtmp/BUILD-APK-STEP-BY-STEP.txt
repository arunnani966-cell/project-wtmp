STEP 1: Deploy Your App Online (Get a URL)
───────────────────────────────────────────

You MUST deploy your app online first to get a URL.

Option A: Netlify (Easiest - 2 minutes)
  1. Go to: https://www.netlify.com/
  2. Sign up (free)
  3. Drag your project folder to Netlify
  4. Wait for deployment
  5. Copy your URL (looks like: https://xxx.netlify.app)

Option B: GitHub Pages
  1. Upload to GitHub
  2. Enable Pages
  3. Get URL

═══════════════════════════════════════════════════════════

STEP 2: Use PWA Builder
─────────────────────────

1. Go to: https://www.pwabuilder.com/

2. In the text box at the top, ENTER your URL:
   Example: https://recipe-generator.netlify.app
   
   ⚠️  DO NOT drag files here!
   ⚠️  DO NOT use localhost!
   ⚠️  Just TYPE the URL!

3. Click the "Start" button

4. Wait for analysis (10-30 seconds)

5. You'll see your app score and options

═══════════════════════════════════════════════════════════

STEP 3: Build APK
──────────────────

1. Click "Build My PWA" button

2. Select "Android"

3. Click "Generate Package" or "Download"

4. Your APK file will download!

═══════════════════════════════════════════════════════════

IMPORTANT NOTES
────────────────

❌ PWA Builder does NOT accept:
   - File paths (C:\Users\...)
   - Localhost URLs (http://localhost:8000)
   - Dragging files

✅ PWA Builder DOES accept:
   - Live web URLs (https://your-app.netlify.app)
   - URLs from Netlify, GitHub Pages, Vercel, etc.

═══════════════════════════════════════════════════════════

QUICK SUMMARY
──────────────

1. Deploy to Netlify → Get URL
2. Go to pwabuilder.com → Enter URL
3. Build APK → Download

That's it!
